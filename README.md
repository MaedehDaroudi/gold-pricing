# 🛠️ gold-pricing


## run 

### ✅ Install dependencies:

```
npm i 
```
 
### ✅ Run with PM2:

```
 pm2 run  dev
 ```

### ✅ Restart pm2:  
  
```
pm2 run restart 
```

### ✅ Stop pm2:  
  
```
pm2 run stop
```  
  
### ✅ See logs:  
  
```
pm2 run logs
```
